A simple Rock–Paper–Scissors game where the user plays against the computer using random choices.

What I learned:
Using the random module for computer decisions 
Implementing game logic with if-elif-else conditions
Handling user input inside a loop
Writing cleaner and more readable code

“This project is a Rock Paper Scissors game     
 developed using Python.
 The user provides an input and the computer   
  randomly selects a choice using the random  
  module.
  I used conditional statements to compare both    
  choices based on predefined game rules.
  The game runs continuously inside a loop until the     
  user decided to quit."

 Tech used:
  Python | Loops | Conditional Statements | 
  Random Module 

This may look small but building projects like this helps strengthen core logic and confidence.
